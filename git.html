How to setup Github

  1. Go to GitHub.com
  2. Create an account or Login.
  3. Click on the plus sign to start a new repository.
  4. Name your repository. Your username will already be included, so use a name that will describe this repository such as WebClass.
  5. Select the option to include a readme file.
  6. When the repository is created, click on settings
  7. Scroll down the page to the GitHub Pages section.
  8. Select "Master branch" from the dropdown menu and click Save.
